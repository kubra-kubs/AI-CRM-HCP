# AI-First CRM HCP Module

An AI-powered Customer Relationship Management (CRM) module that helps healthcare professionals (HCPs) log and manage interaction details using natural language conversations.

## Project Overview

This project implements an AI-first interaction logging system where users can describe an HCP interaction through a chat interface. The AI extracts structured information and automatically fills the interaction form while saving the data into PostgreSQL.

## Features

- AI-powered interaction logging
- Natural language conversation interface
- Automatic extraction of HCP interaction details
- Auto-populated read-only interaction form
- PostgreSQL database integration
- FastAPI backend APIs
- React frontend dashboard
- Groq LLM integration
- LangGraph tool-based architecture
